### ☑️ 고전게임 갈스패닉을 모작한 게임입니다.

- Window GDI 함수를 사용하여 배경의 마스크 효과를 적용했습니다.
- 이론적으로 배웠던 윈도우 프로그래밍, GDI 등에 관하여 익숙해질 수 있었습니다.
